## 🚀 Overview
**Xcore** is a fully dynamic restaurant application designed to provide a seamless ordering experience. The app's unique selling point is its **Dynamic Category Engine**, which automatically generates the user interface based on the data present in the Firestore database.

## ✨ Key Features
- **🎯 Fully Dynamic UI**: Categories and menu items are fetched and grouped in real-time from Firebase.
- **🎨 Modern Dark Theme**: A sleek, user-friendly interface optimized for night mode.
- **⚡ Real-time Updates**: Instant synchronization with the database for pricing and availability.
- **🛍️ Integrated Cart**: A fluid shopping cart system with meal customization options.
- **📏 Responsive Design**: Optimized for various screen sizes with no overflow issues.

## 🏗️ Architecture
The project follows a clean **Model-View-Provider** architecture to ensure maintainability:
- **Models**: Handles data transformation (Firebase Map to Object).
- **Views**: Optimized widgets like `CategorySelector` and `MenuItemCard`.
- **Logic**: Centralized in `MenuUtils` for dynamic category generation.
